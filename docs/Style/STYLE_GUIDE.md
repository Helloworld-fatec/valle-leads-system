# 🎨 STYLE GUIDE — Valle Leads System

## 🎯 Identidade Visual

Sistema de gestão de leads automotivos com foco em organização, agilidade e conversão.
Interface moderna, limpa e pensada para uso interno por equipes de vendas.

---

## 🎨 Paleta de Cores

### 🔵 Cores principais

* Cor primária: #1E3A8A
* Cor secundária: #2563EB

### ⚪ Fundo

* Fundo principal: #F9FAFB
* Fundo secundário: #FFFFFF

### 📝 Texto

* Título: #111827
* Texto normal: #6B7280

### ✨ Destaque

* Cor de destaque (accent): #2563EB

### 🚨 Estados

* Sucesso: #16A34A
* Erro: #DC2626

## 🖼️ Logo

### Conceito

O logo representa a jornada completa do lead:
do primeiro contato (A) ao fechamento (Z).

### Cores

* Azul (#1E3A8A): confiança e início
* Vermelho (#DC2626): conversão e venda

### Variações

* Logo principal (azul + vermelho)
* Logo branco (para fundo escuro)
* Logo reduzido (somente símbolo AZ)

### Caminho

`src/assets/logo.svg`
